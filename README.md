# Goal

This repo provides examples of using Azure Resource Manager (ARM) method to provision 'Grounding with Bing custom search' resource and configuration.

# REST API

cURL examples are included to demonstrate using ARM REST APIs to provision Bing resources and create or update allowed and blocked domain configurations.

# Python SDK

bing_custom_search_arm.py is an example of Python code which uses ARM method to login, authenticate, create Bing resource and configure/update Bing inclusion/exclusion list.

# C# SDK

C# code is provided in a separate folder which uses ARM method to login, authenticate, create Bing resource and configure/update Bing inclusion/exclusion list.
