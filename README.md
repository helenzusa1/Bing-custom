# Goal

This repo provides examples of using ARM method to provisioning 'Grounding with Bing custom search' resource and configuration.

# REST API

cURL examples are included to demonstrate using Azure Resource Manager (ARM) REST APIs to provision Bing resources and create or update allowed and blocked domain configurations.

# Python SDK

bing_custom_search.py is an example of Python code which uses ARM method to login, authenticate, create Bing resource and configure/update Bing inclusion/exclusion list.

# C# SDK

C# code is provided in a separate folder which uses ARM method to login, authenticate, create Bing resource and configure/update Bing inclusion/exclusion list.
